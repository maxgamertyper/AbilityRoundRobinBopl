# AbilityRoundRobinBopl
A bopl mod to change the way abilities are picked up

Example [Here](https://github.com/maxgamertyper/AbilityRoundRobinBopl/blob/main/AbilityRoundRobin.mp4) and [Here](https://youtu.be/Ved8MAjPTDA)

## caveat
This mod is incompatible with the mod AbilityStorm due to a function override inside of AbilityStorm, causing my transpiler to break
